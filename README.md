A MERN stack based project for a social media application. It is a fully working and responsive application.  
MongoDB, Express, React, Redux, Node.js are used in this project.

---

## Features

### User Features
- Register and login users.
- Post images using camera or file system.
- Pagination on every page.
- Dark mode.
- Copy link of post.
- Report posts for spam.
- Search users by username.
- User suggestions menu.
- Save posts to collections.
- View saved posts page.
- Delete posts and comments.
- Explore page to view posts by random users.
- Notifications page.
- Profile page and edit profile page.
- Passwords are stored in encrypted format with salt.
- Create, edit, like, comment, share posts.
- Reply to comments and like comments.
- Clear notifications option.
- Profile page shows user details, posts, followers, and following.

### Admin Features
- Admin panel shows total posts, users, reported posts, etc.
- Admin can create or assign other admin accounts.
- Admin can see posts reported by multiple users.
- Admin can delete reported posts.

## Requirements

- Node.js
- MongoDB or Mongo Atlas
- Cloudinary account
- NPM

---

**How to run this application**
1.  Make sure MongoDB is running on your system or online.
2. include MongoDB database link in .env file.
3. Add cloudinary data in /client/src/utils/imageUpload.js file.
4. Clone this repository
5. Open command line in the cloned folder,
    - To install dependencies for backend, run  `npm install` in main folder.
    - To run backend type command `node server` in main folder.
    - To install dependencies for frontend , run  `npm install` in `/client` folder.
    - To run frontend type command `npm start` in `/client` folder.
6.  Open  [localhost:3000](http://localhost:3000/)  in the browser.
 **And application will be up and running.**
